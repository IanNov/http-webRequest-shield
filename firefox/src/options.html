HTTP Request Shield

HTTP webRequest protection

You can whitelist hosts here by simply adding hostname (eg. example.com) to stop them from being blocked by HTTP webRequest protection. Write down the hostname itself (excluding "www", but including all other domains eg. ".com"). We generally do not recommend doing that for hostnames from public internet. Please note, that these hostnames are checked with domain hierarchy, so whitelisting example.com does also whitelist news.example.com.
